# Rock-Paper-Scissors-Game
-----

Built With
-----

* HTML
* CSS
* JAVASCRIPT

Instrucions to Play
----

* Choose between Rock, Paper or Scissors by clicking on the icon.
* Rock beats Scissors.
* Scissors beats Paper.
* Paper beats Rock.
* Player with highest score wins.
* Play again!
